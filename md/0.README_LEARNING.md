## 💡 学习建议

### 对于前端开发者的特别提示

1. **思维转换**

   - React/Vue 组件 ≈ Compose 组件（@Composable）
   - CSS/样式对象 ≈ Modifier
   - useState ≈ remember + mutableStateOf
   - 路由配置 ≈ AndroidManifest.xml + Navigation

2. **开发工具**

   - Android Studio 是必需的（用于预览、运行、调试）
   - Cursor 可以用于代码编写（利用 AI 能力）
   - 两者可以同时使用，但要注意文件同步

3. **学习顺序**
   - 先理解 Kotlin 基础语法（和 JS 很相似）
   - 然后学习 Compose（和 React 很相似）
   - 最后学习 Android 特有概念（Activity、生命周期等）

## 🎯 核心概念速记

| Android/Kotlin            | 前端对应          | 说明                |
| ------------------------- | ----------------- | ------------------- |
| Activity                  | Route/Page        | 应用的一个屏幕/页面 |
| @Composable               | React Component   | 可组合的 UI 组件    |
| Modifier                  | className + style | 样式和布局修饰      |
| remember + mutableStateOf | useState          | 状态管理            |
| Column/Row                | flex-col/flex-row | 布局组件            |
| MaterialTheme             | Theme Provider    | 主题系统            |
| build.gradle.kts          | package.json      | 依赖和构建配置      |
| res/values/strings.xml    | i18n 文件         | 字符串资源          |
| AndroidManifest.xml       | 路由配置 + 权限   | 应用清单            |
